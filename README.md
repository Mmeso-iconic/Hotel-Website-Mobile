# Hotel Mobile Website

This is a simple and responsive hotel-themed website that presents Hotel Bliss – a modern, elegant, and welcoming destination for travelers. It includes a clean layout, room highlights, and a mobile-friendly design. This project was built using HTML, CSS, and JavaScript to practice layout structuring, styling, and basic interactivity.

## Introduction

This project is a hotel website layout created to practice building visually engaging and interactive web pages using HTML, CSS, and JavaScript. The site showcases sections such as the hotel’s intro, rooms, facilities, testimonials, and a contact form. The goal was to create a clean, responsive interface suitable for a real-world hospitality brand

## Features

- Responsive layout (works on desktop and mobile)
- Hero section with call-to-action
- Room gallery showcasing different suites
- Facilities section with hotel services
- Guest testimonials
- Contact form and location info
- Custom fonts and consistent styling
- Basic navigation toggle for mobile

---

## Technologies Used

- **HTML5** – Page structure and semantic markup
- **CSS3** – Styling, responsive layout, and animations
- **JavaScript** – Navigation toggle and interactivity
- **Google Fonts** – "Montserrat" for modern typography

## Setup and Installation

1. Download or clone this repository.
2. Open the folder in your code editor.
3. Make sure any necessary assets (images, icons, etc.) are in the right folders.
4. Open index.html in your browser to view the website.

## Usage

You can use this layout as a hotel landing page, a starting point for hospitality web design, or a personal practice project. Feel free to replace the text, images, and scripts with your own to customize the design.
